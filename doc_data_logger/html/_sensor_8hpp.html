float generateFakeData ( float  minValue,
float  maxValue,
uint32_t  currMicros,
float  offset = 0,
uint32_t  period = 5000000 
)

Generate some fake data; useful for debugging.

Compute \( \texttt{mid} = \cfrac{\texttt{minValue} + \texttt{maxValue}}{2} \) and \( \texttt{amplitude} = \cfrac{\texttt{maxValue} - \texttt{minValue}}{2} \)

Pseudo-random output: \( \sin(2\pi * \texttt{currMicros} / \texttt{period}) * \texttt{amplitude} + \texttt{mid} + \texttt{offset} \)

Parameters
minValue: Minimal possible value
maxValue: Maximal possible value
currMicros: Current time, in microseconds
offset: Offset from the mid value
period: Temporal period of the sin function used to compute output (in microseconds)
Returns
float : A pseudo-random number
